# Archer Engine

A game engine for building old style text adventure.
Design your world in json, put scenes, items and beings in it, and play infinite worlds powered by glorious text.

## Features

* 2d and linear maps
* Command alias

ex:
```json
"customCommands" : [
    { "alias":"What's inside?" , "mapTo" : "Look", "param":""} 
  ]
```
